변수(variable) - 데이터를 하나 저장하는 공간

변수 타입
	primitive type(기본형) - byte int char float double boolean
		-> 할당되는 크기가 정해져 있음, 저장할수 있는 데이터의 범위가 있음
		
	reference type(참조형) - String .....
		-> 할당되는 크기가 미리 정해져 있지 않음 --> 그때 그때 마다 크기가 달라진다.
		
기본형
	정수형 - byte(1) short(2) int(4) long(8)
	  -> 문자형 - char(2)
	  
	실수형 - float(4) double(8)
	
	논리형 - boolean(1)
	
변수명 짓는 규칙
	1. 알파벳 대소문자, 숫자, _ $ 조합으로 변수명을 짓는다.
	2. 첫 글자는 숫자가 올수 없다.
	3. 키워드 사용 불가
	4. 중복된 변수명 사용할수 없음
