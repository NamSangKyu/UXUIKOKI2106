상수
	변하지 않는 값 ---> pi
	코드상에서 사용하는 값
	int num = 10; // 10이 상수
	String name = "남상규";//남상규 <-- 상수
	final 키워드 --> 해당 변수를 상수로 만들때
	final double pi = 3.1415;

데이터 형변환
	기존에 선언시 또는 사용한 데이터의 타입을 임시로 다른 타입으로 바꾸는 행위
	
	자동 형변환 : 연산을 하면서 자동으로 형변환이 되는 경우
	강제 형변환 : 사용자가 원하는 타입으로 형변환을 하는 경우
	
삼항 연산자
	조건식 ? 참일때 나타낼 식 : 거짓일때 나타낼 식
	
반복문(Loop)
	연속적으로 반복된 동일한 코드를 묶어서 표현
	원하는 만큼 코드 수행을 반복 시켜주는 문법(조건식의 결과가 true일때만 반복)
	while, do-while, for
	
분기문
	조건식에 따라서 실행할 코드를 다양하게 실행
	if, switch
	
break, continue
	break : 반복문에서 break 키워드를 사용하면 
			break를 직접 감싸고 있는 반복문을 강제종료
	
	continue : 반복문에서 데이터 계산시 문제가 발생할수 있는 데이터를 발견하면
			   더이상 연산하지 않고 다음 반복으로 넘어가는 키워드
