상속(Inheritance)
	: 부모 클래스가 자식 클래스에게 가지고있는 변수와 메서드를 물려주는 행위
	
	상속의 장점 : 부모가 준 코드를 쓸수 있음(재사용)
			   부모와 자식간의 관계를 만들어줌 --> 자식 클래스가 부모 클래스로 형변환 
			   --> 부모 클래스로 모든 자식클래스를 묶어서 실행이 가능
			   --> 자식 만약에 부모가 준 메서드를 재정의 한다면 현재 타입이 부모라도 원본 객체인 자식이 재정의한 메서드를 실행
	자바에서 직접적으로 상속을 받는 클래스는 한개
	
	
Player 클래스
	스킬, hp, 스킬실행, 캐릭터사망
	
Skill 클래스
	스킬활성화(플레이어)
