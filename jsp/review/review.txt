jsp --> java server page
	jsp/Servlet --> 하나
	
프로젝트 폴더
	src
		main
			webapp : 홈페이지 표현에 필요한 앞단 파일이 있음
					 jsp, css, js, jquery, 이미지
				WEB-INF : 외부에서 접근이 불가한 폴더
					lib : 실제 서비스를 할때 필요한 라이브러리 파일(jar)를 올려놓는 공간 --> Maven
					

주요 객체
	session : 브라우저 영역에서 저장하고있는 객체 --> 브라우저를 종료할때까지, 로그인 및 개인정보 활용하는 내용
	request : 사용자가 요청한 데이터를 담고 있음, 서버에서도 사용자에게 응답할 데이터를 저장 가능
	response : 사용자에게 응답하는 객체, 사용자에게 전달할 내용을 스트림으로 전달
	
Ajax 응답시 : response.getWriter() --> 사용자에게 내용을 전달할떄 쓰는 스트림
			println, write 메서드를 이용하여 전달
			

페이지 이동 방법
	sendRedirect : 사용자의 요청을 더이상 받지 않을떄, request 정보가 전부 소멸
	forward : 사용자의 요청을 계속 받을때, request 정보가 소멸되지 X
