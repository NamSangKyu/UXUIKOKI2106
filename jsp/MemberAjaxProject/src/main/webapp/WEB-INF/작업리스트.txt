1. index.jsp 만들어서 member_manager.jsp로 이동
   모든 회원 정보를 list에 넣어서 member_manager.jsp가
   로드시 전부 출력
   
2. 등록 버튼을 누르면 insertMember.do를 호출해서 입력한 회원 정보를 등록
   회원 목록을 최신화 결과를 Ajax로 받아서 처리	
   
3. 회원 목록 끝에 수정 삭제 버튼을 추가
   수정 class : btnUpdate
   삭제 class : btnDelete
   
4. 삭제 버튼 눌렀을 때 해당 데이터를 삭제 처리
   삭제 처리는 Ajax로 하고 회원 목록을 최신화 Ajax로 처리
   	--> 테스트 반드시 연속적으로 2건이상 눌러서 테스트
   	--> 완성 후 두번째 부터 삭제 작업이 동작하지 않음 --> 동작하게끔 작업
